Html

L'Html è un contenitore che viene utilizzato per contenere una pagina di codice scritto in html 
            
                <html lang="en">
                <head>
                    <meta charset="UTF-8">
                        <meta http-equiv="X-UA-Compatible" content="IE=edge">
                        <meta name="viewport" content="width=device-width, initial-scale=1.0">
                        <title>Document</title>
                    </head>
                    <body>
                    
                    </body>
                </html>

Head

head è un contenitore di metadati e si utilizza per eventuali collegamenti ad altri fogli di codice (css) 
            
                
                <head>
                    <meta charset="UTF-8">
                    <meta http-equiv="X-UA-Compatible" content="IE=edge">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Document</title>
                    <link rel="stylesheet" href="style.css">
                </head>

Body

Body è il corpo della pagina. Contiene la maggior parte delle informazioni visibili nella nostra pagina web 
            
                
                <body>
                    <ol>
                        <li>
                            <a href=""/"https://www.digitazon.school/" target="_blank">Digitazon School</a>
                        </li>
                        <li>
                            <p> ciao sono un altro paragrafo </p>
                        </li>
                    </ol>
                </body>

Tag testi

p è un paragrafo
br si utilizza per andare a capo
i tag h1/h2...h6 sono titoli/sottotitoli di varia grandezza e importanza
u rende il testo sottolineato
b e strong rendono il testo grassettato, strong ha una maggiore importanza semantica
i rende il testo in corsivo
quote(per le citazioni in line) e blackquote si utilizzano per le citazioni e il secondo consente di inserire l'autore
a serve per rendere un testo un link
del serve per rappresentare un testo errato(?)
sub serve per mettere un testo su un piano inferiore
sup serve per mettere un testo su un piano superiore

Img

img serve per inserire un'immagine all'interno della pagina 
             
                <div>
                    <img src="esempio.jpg">
                </div>

Liste

ol serve per creare liste numerate
ul serve per creare liste non numerate
li rappresenta i dati all'interno della lista 
             
                <ol>
                        <li>elemento 1</li>
                        <li>elemento 2</li>
                        <li>elemento 3</li>
                        <li>elemento 4</li>
                </ol>

Tabella

table è il tag per definire la tabella
th è il tag per definire le celle più in alto della tabella
tr è il tag chiamato table row e rappresenta la riga della tabella
td è il tag table data e rappresenta i dati all'interno delle celle della tabella
tbody è il tag table body e rappresenta il corpo della tabella 
            
                <table>
                    <tr>
                        <th>Classe</th>
                        <th>Matricola</th>
                        <th>Locazione</th>
                    </tr>
                    <tr>
                        <td>1A</td>
                        <td>12</td>
                        <td>Napoli</td>
                    </tr>
                    <tr>
                        <td>2A</td>
                        <td>34</td>
                        <td>Milano</td>
                    </tr>
                </table>

Div

Il div è un contenitore generico ed è un elemento di tipo block level element 
                
                    <div>
                    <h1>Glossario tag</h1>
                    <button>Cliccami</button>
                    <div>

Struttura Web Page

header rappresenta la parte superiore della pagina
main rappresenta la parte centrale della pagina (deve esisterne solo uno)
footer rappresenta la parte più in basso della pagina 
                
                    <header>
                    <h1>Glossario tag</h1>
                    </header>
                    <main>
                        <div>
                            <img src="immagine.jpg">
                            <button>Cliccami</button>
                    </main>
                    <footer>
                    <p><a href="#about.html">Contattami</a></p>
                    </footer>

Section

section rappresenta una sezione generica della pagina 
                
                    <section>
                        <img src="immagine.jpg">
                        <i>Testo fantastico sotto immagine</i>
                    </section>
